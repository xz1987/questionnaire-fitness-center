# 平板访问指南

## 📱 从平板设备访问调查问卷

### 前提条件

1. **服务器电脑和平板设备必须在同一Wi-Fi网络**
2. **服务器必须正在运行**
3. **平板设备需要有浏览器**（Safari、Chrome、Edge等）

### 步骤

#### 1. 启动服务器

在Windows电脑上：
- 双击 `start.bat` 启动服务器
- 或者运行 `node server.js`

#### 2. 查看网络IP地址

服务器启动后，控制台会显示：

```
============================================================
Server running successfully!
============================================================
Local access:    http://localhost:3000
Network access:  http://192.168.1.100:3000
============================================================

To access from tablet/phone:
1. Make sure your device is on the same network
2. Open browser and go to: http://192.168.1.100:3000
============================================================
```

**记下 "Network access" 后面的地址**（例如：`http://192.168.1.100:3000`）

#### 3. 在平板上访问

1. **确保平板连接到同一Wi-Fi网络**
2. **打开浏览器**（Safari、Chrome、Edge等）
3. **输入网络IP地址**（例如：`http://192.168.1.100:3000`）
4. **开始填写调查问卷**

### 故障排除

#### 问题：无法连接到服务器

**检查清单：**
- [ ] 服务器是否正在运行？
- [ ] 平板和服务器是否在同一Wi-Fi网络？
- [ ] IP地址是否正确？
- [ ] Windows防火墙是否阻止了连接？

**解决方案：**
1. **检查防火墙**：
   - Windows可能会阻止Node.js的网络访问
   - 首次运行时，Windows会询问是否允许访问
   - 选择"允许访问"

2. **手动允许防火墙访问**：
   - 打开"Windows Defender 防火墙"
   - 点击"允许应用通过防火墙"
   - 找到Node.js，勾选"专用"和"公用"

3. **检查IP地址**：
   - 确保使用"Network access"显示的IP地址
   - 不要使用"localhost"或"127.0.0.1"

#### 问题：触摸拖拽不工作

**解决方案：**
- 确保使用现代浏览器（Chrome、Safari、Edge）
- 尝试长按项目，然后拖动
- 如果仍然不工作，可以使用"×"按钮移除，然后重新选择

#### 问题：页面显示不正常

**解决方案：**
- 清除浏览器缓存
- 尝试刷新页面（F5或下拉刷新）
- 尝试使用其他浏览器

### 测试建议

1. **先在电脑上测试**：确保服务器正常运行
2. **在同一网络测试**：确保平板和服务器在同一Wi-Fi
3. **测试所有功能**：
   - 单选按钮
   - 复选框
   - 排名拖拽
   - 文本输入
   - 表单提交

### 安全提示

⚠️ **注意**：服务器监听所有网络接口（0.0.0.0），这意味着同一网络上的任何设备都可以访问。

- ✅ **适合**：家庭网络、办公室内部网络
- ⚠️ **不适合**：公共Wi-Fi、不安全的网络

如果需要在公共网络使用，建议：
- 使用VPN
- 或者修改服务器只监听特定IP地址
